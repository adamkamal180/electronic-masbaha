## 🚀 المميزات التقنية
* **نظام الأهداف الذكي:** إمكانية الاختيار بين أهداف محددة (33، 99) أو نظام العد المفتوح (∞).
* **مصفوفة النشاط (Heatmap):** تتبع سجل تسبيحاتك في آخر 35 يوماً بصرياً لتقييم مدى الالتزام.
* **تفاعل حسي وصوتي:** دعم الاهتزاز (Vibration) والتنبيهات الصوتية عند الوصول للأهداف المحددة.
* **حفظ البيانات:** تخزين محلي (LocalStorage) يضمن بقاء أرقامك وسجلاتك حتى بعد إغلاق الصفحة.
* **واجهة مخصصة:** تصميم عصري بالألوان (الأبيض، الأسود، البرتقالي) مع دعم العبارات المخصصة.
